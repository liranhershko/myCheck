# exercise1
### Server
1. Make sure that mongo is running in Terminal.
1. Go to the server dir.
1. Run 
```
$ npm install
$ npm run dev
```
server should listen on port 3090.

### Client
1. Go to the client dir.
1. Run 
```
$ npm install
$ npm start
```
Open http://localhost:8080/
...
and that's pretty much it!
